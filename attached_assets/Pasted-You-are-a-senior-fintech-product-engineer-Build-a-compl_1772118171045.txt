Build a complete, production-ready Mobile & DTH Recharge application named "RupyaSetu".

TECH STACK:
- Frontend: React Native (Expo) + TypeScript
- Backend: Node.js + Express
- Database: MongoDB
- Auth: OTP based login (mobile number)
- API integration: Paysprint Recharge API (UAT)

DESIGN REQUIREMENTS:
- Professional fintech app like PhonePe / Google Pay
- Clean, modern UI
- Primary color: #2E9E5B
- Font: Inter
- Rounded cards, subtle shadows
- Bottom tab navigation
- Smooth animations

PAYMENT FLOW (TEMPORARY / MANUAL MODE):
- When user clicks "Pay Now", open installed UPI apps using UPI Intent
- Pass fixed recharge amount, payee UPI ID, and note
- User completes payment in UPI app
- User is redirected back to RupyaSetu app
- App shows a screen to manually enter UPI Reference Number (UTR)
- Backend stores UTR with transaction as "PAYMENT_UNVERIFIED"
- Recharge API call is triggered only after basic validation of UTR format
- This payment flow is configurable via environment variable:
  PAYMENT_MODE=MANUAL or PAYMENT_MODE=GATEWAY
- Code should be structured so that payment gateway can be added later without refactoring

FEATURES:
1. OTP based login using SMS API

10. Environment variable support
11. Error handling & loading states

PAYS PRINT CONFIG (USE ENV VARIABLES):
